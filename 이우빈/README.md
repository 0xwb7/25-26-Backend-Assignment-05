# 25-26-Backend-Assignment-05

## 회원가입

- 두 가지 방식으로 회원가입 가능
    - 직접 회원가입
    - 구글 oauth를 통한 가입

## 로그인

- 로그인 시, 어떤 경로로 회원가입을 했는지 구분

## 글 작성

- 회원가입을 한 모든 유저는 글 작성이 가능하다.

## 댓글 작성

- 회원가입을 한 모든 유저는 댓글 작성이 가능하다.

## 글 및 댓글 삭제

- 본인이 작성한 글과 댓글만 삭제 가능하다.
- Admin은 모든 글과 댓글 삭제가 가능하다.

## Admin Page

- Admin 계정으로 로그인하면, Admin Page가 redirect된다.
- Admin Page에서는 유저, 글, 댓글 관리가 가능하다.
